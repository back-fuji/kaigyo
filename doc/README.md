# ドキュメント

このディレクトリには、Flutter開発に関するドキュメントを配置しています。

## ドキュメント一覧

### [Flutter SDK リファレンス](./flutter_sdk_reference.md)

Flutter SDKでよく使う機能をまとめたリファレンスです。

- 基本Widget（MaterialApp、Scaffold、Text など）
- レイアウトWidget（Column、Row、Stack など）
- 入力Widget（TextField、TextEditingController など）
- ボタンWidget（ElevatedButton、IconButton など）
- ナビゲーション（Navigator、PopScope など）
- 状態管理（StatefulWidget、setState など）
- ライフサイクル（initState、dispose など）
- 非同期処理（Future、async/await など）
- よく使うクラス（BuildContext、ThemeData など）
- よく使うメソッド（runApp、WidgetsBinding など）

### [Flutter開発ガイド](./flutter_development_guide.md)

Flutter開発で覚えるべき知識をまとめたガイドです。

- Dart言語の基礎（型、関数、クラス）
- 変数宣言（var、final、const、late など）
- 非同期処理（Future、async/await、Stream）
- Widgetの基本概念（StatelessWidget、StatefulWidget）
- 状態管理のパターン
- ライフサイクル
- アーキテクチャパターン
- パフォーマンス最適化
- エラーハンドリング
- ベストプラクティス

## 使い方

開発中に分からないことがあったら、これらのドキュメントを参照してください。

- **Widgetの使い方が分からない** → [Flutter SDK リファレンス](./flutter_sdk_reference.md)
- **Dartの構文が分からない** → [Flutter開発ガイド](./flutter_development_guide.md)
- **ベストプラクティスを知りたい** → [Flutter開発ガイド - ベストプラクティス](./flutter_development_guide.md#ベストプラクティス)
